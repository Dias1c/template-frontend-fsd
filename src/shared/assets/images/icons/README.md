# icons

## What stored in this directory

Here stored icon files

## Naming of icons

1. Name of icons must follow this pattern: `[name]-[style]-[color].[extension]`

### Pattern

- `[name]` - icon name
- `[style]` - style of icon
- `[color]` - icon color (`dark`, `gray`, `red`, `light`, `white` etc)
- `[extension]` - extension (`svg`)
